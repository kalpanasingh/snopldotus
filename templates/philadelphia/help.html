Viewing Reports

To view a report, click its ID in the list. Note that submitted (finished) reports are marked with a and reports still in progress are marked with a .

Reports are arranged as a time-ordered series of "blocks" - customizable templates where the shift operator fills in information.

Blocks with less-common information are hidden by default. You can toggle showing/hiding any block by clicking the arrows:

Hide

To download the report as a (LaTeX) PDF, click the save icon:
Save as PDF

To return to the index page, you can click the back button:
Back to home
or choose "Detector > Shift Reports" from the menu bar.

Creating Reports

To create a new report, click the New Report button:
Create new report

This will open the Shift Report Editor, where you will will compose your report out of time-ordered "blocks" - customizable templates where you fill in information. Each block contains a set of fields -- text boxes, checkmarks, etc. A few recommended blocks are provided for you.

You can add and remove blocks, and customize any block by adding and removing fields. Do not remove the Basic Information block, as this contains information used in the listing page.

You'll encounter a few simple buttons:
Remove field (next to field name)
Remove block (upper right of block)
Add block
Submit report
Show
Hide
Add field Add a new field to the block
Add attachment Attach a file to the block

It's possible that while you're editing, someone else will come along and start editing one of your blocks. Only one person can modify a block at a time, so the block will be hidden and shown with a lock: . You can take back control by clicking the lock, but should communicate with the other party (consider using the comments - a notification is shown when new comments are made) to coordinate your efforts.

Editing Reports

Reports marked as in-progress with a can be edited (while submitted reports, marked with a , cannot).

To edit a report, open it in the viewer and click the edit icon:
Edit

Each editor session has a unique Editor ID, and only one editor at a time can modify a shift report block. Since this new editor doesn't "own" any of the blocks, they will all appear as locked: .

You can take over control of a block, which will lock out the block for all other editors, by clicking the unlock button:
Take control of block

This loads the latest version of the block and allows you to edit it. The lock button now appears as the show/hide buttons which work the same as in the report viewer.

You may also add new blocks to the report, and remove blocks which you have editor control of. These changes will appear in other editors along with a notification popup.

For an overview of the editor controls, see the Creating Reports section.

Making Comments

Whether you are viewing or editing a report, you can make comments and everyone who's watching the report (viewing or editing) will be notified.

At the bottom of the view/edit page, enter your comment and your name and click "Submit."
